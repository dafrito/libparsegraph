parsegraph_user_httpd

User functions for use with Apache httpd modules.

parsegraph_authenticate(request_rec* r, struct parsegraph_user_login* createdLogin)

Authenticates the cookie provided in the given request, storing the result of authentication in createdLogin.

Transactional.

  • Returns parsegraph_OK if the user was authenticated.
  • Returns parsegraph_ERROR if a serious error occurs during authentication.
  • Returns parsegraph_SESSION_DOES_NOT_EXIST if the session was not given.
  • Returns parsegraph_SESSION_DOES_NOT_MATCH if the session is invalid.

apr_status_t parsegraph_removeSession(request_rec* r);

Removes the session cookie from the given request.

HttpOnly;Max-Age=315360000;

apr_status_t parsegraph_setSession(request_rec* r, struct parsegraph_user_login* createdLogin);

Sets the session cookie for the given request to the given login.
